# ============================================================================
# DOWNLOAD AND UPLOAD PROCESS
# ============================================================================

# STEP 1: Download Images
# 1. Go to Unsplash.com or Pexels.com
# 2. Search for each product type
# 3. Download high-resolution images
# 4. Rename files according to your products

# STEP 2: Upload to Cloudinary
# 1. Login to your Cloudinary dashboard
# 2. Create folder structure:
#    - jewelry/rings/
#    - jewelry/necklaces/
#    - jewelry/earrings/
#    - jewelry/bracelets/
#    - jewelry/pendants/
#    - jewelry/sets/
# 3. Upload images to appropriate folders
# 4. Copy the URLs generated by Cloudinary

# STEP 3: Update SQL File
# 1. Open update_product_images.sql
# 2. Replace 'YOUR_CLOUDINARY_CLOUD_NAME' with your cloud name
# 3. Replace image paths with your actual paths
# 4. Run the SQL in Supabase

# ============================================================================
# CLOUDINARY OPTIMIZATION TIPS
# ============================================================================

# AUTO-OPTIMIZATION:
# Cloudinary can automatically optimize images:
# - Add /f_auto,q_auto/ to URLs for automatic format and quality
# - Example: https://res.cloudinary.com/yourcloud/image/upload/f_auto,q_auto/v1/jewelry/rings/ring1.jpg

# RESPONSIVE IMAGES:
# Create multiple sizes for different devices:
# - Thumbnail: /w_300,h_300,c_fill/
# - Medium: /w_600,h_600,c_fill/
# - Large: /w_1200,h_1200,c_fill/
